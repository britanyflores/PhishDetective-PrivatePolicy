PhishDetective Privacy Policy

Last updated: October 23, 2025

Information Collection and Use

PhishDetective is a Chrome extension that helps users identify potentially unsafe or phishing websites by scanning URLs using the Google Safe Browsing API.

PhishDetective does not collect, store, or share any personal information, browsing history, or website content. When a user manually initiates a scan, the extension sends the current tab’s URL to a secure backend server, which relays it to the official Google Safe Browsing API to determine if the site is associated with known threats.

The scanned URLs are used only for this safety check and are not logged, profiled, or linked to any user or identifiable information.

Data Sharing

The only external service used is Google Safe Browsing, accessed via the extension’s backend API. No data is sold, shared, or transferred to any third parties for marketing, analytics, or advertising purposes.

Security

All network communications between the extension, the backend service, and Google Safe Browsing are encrypted using HTTPS to ensure the privacy and integrity of transmitted data.

User Control

Users must manually initiate scans. The extension performs no background monitoring, tracking, or automatic data collection.
